# TechMatch – Smart Electronics Store

##  Project Overview
TechMatch is a smart electronics e-commerce website that allows users to browse and purchase electronic devices easily.

##  Features
- Browse products (phones, laptops, accessories)
- Filter products by category and price
- Product details page
- Shopping cart system
- Admin dashboard for managing products

##  Technologies Used
- HTML
- CSS
- JavaScript
