Group 9MS02 · BB Group #4 · Term 2 (2025–2026)

---

## 📌 Project Overview

TechMatch is a smart electronics e-commerce website that allows users to browse, explore, and purchase devices بسهولة من خلال واجهة حديثة ونظيفة.

The system focuses on simplicity, usability, and a structured shopping experience.

---

## 🚀 Features

### 👤 Customer Side
- Browse products (Phones, Laptops, Tablets, Accessories)
- Filter products by category and price
- View product details
- Add and manage shopping cart
- Checkout process (shipping + payment)
- Contact page with Google Maps

### 🔐 Admin Side
- Admin login system
- Add new products (with image upload)
- Modify existing products
- Delete products
- Search products
- Dashboard overview
