📌 Project Overview

TechMatch is a smart electronics e-commerce website that allows users to browse and purchase electronic devices easily. The platform provides a clean UI and interactive filtering system to enhance the shopping experience.

🚀 Features
Browse products (phones, laptops, tablets, accessories)
Filter products by category and price
Product details page
Shopping cart system
Admin dashboard for managing products
Responsive modern UI design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
